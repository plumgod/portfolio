QUICK START GUIDE
=================

1. Download and unzip the 'austin-portfolio' folder

2. Open Terminal and navigate to the folder:
   cd path/to/austin-portfolio

3. Install dependencies (first time only):
   npm install

4. Start the dev server:
   npm run dev

5. Open browser to: http://localhost:3000

6. To stop the server: Press Ctrl+C in terminal


EDITING THE SITE
================

Open any .tsx file in your text editor to make changes:
- app/page.tsx - Homepage
- app/research/page.tsx - Research page  
- app/music/page.tsx - Music page
- app/about/page.tsx - About page

Changes auto-reload in browser while dev server is running.


ADDING YOUR PHOTOS
==================

1. Create a 'public' folder in the root:
   mkdir public

2. Put your photos there (e.g., public/hero.jpg)

3. Use them in your code:
   <img src="/hero.jpg" alt="Description" />


DEPLOYING TO VERCEL (FREE)
===========================

1. Push your code to GitHub
2. Go to vercel.com and sign up
3. Click "New Project" and import from GitHub
4. Vercel will auto-deploy on every push

Your site will be live at: your-project.vercel.app
Then connect your custom domain in Vercel settings.
